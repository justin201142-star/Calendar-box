# Calendar-box
# 项目日历 (Project Calendar)

这个项目是一个 Web 日历 / 项目管理应用，功能包括：

- 月视图 & 日格视图  
- 任务创建 + 拖拽移动  
- 任务颜色标签  
- 导出当天任务到 `.ics`（iCal）格式，可导入 Google 日历 / 手机日历  

---

## 技术栈

- React  
- Framer Motion  
- Tailwind CSS  

---

## 使用方式

1. 克隆仓库 `git clone https://github.com/你的用户名/项目日历.git`  
2. 安装依赖 `npm install`  
3. 启动开发服务器 `npm run dev`（或 `npm start`，取决于你的构建工具）

---

## 导出 `.ics`

点击页面底部的按钮 “导出本月到 .ics” 就可以下载当前月份任务的 iCal 文件，用于添加到移动设备日历或 Google Calendar。
